# Balanced-Allocations-Study
This repository contains the implementation and experiments for the randomized allocation strategies study.

## Files Description

- `balanced_allocations.py` - Basic strategies: one-choice, two-choice, (1+Î²)-choice
- `d_choice.py` - D-choice strategies for d = 1,2,3,4,5
- `batched_allocations.py` - Batched allocation strategies
- `partial_info_allocations.py` - Partial information strategies with 1 or 2 queries

#How to run
# Basic allocation strategies
python balanced_allocations.py

# D-choice strategies  
python d_choice.py

# Batched allocations
python batched_allocations.py

# Partial information strategies
python partial_info_allocations.py

#OUTPUTS
1.Print experimental results to console
2. Generate corresponding plots in the current directory
